## SDK and MCP

- [Stitch SDK](https://github.com/google-labs-code/stitch-sdk) - Official Node.js SDK for generating screens, editing, creating variants, and extracting HTML/screenshots.
- [Stitch MCP Server](https://github.com/davideast/stitch-mcp) - Community-built MCP server for integrating Stitch with Claude Code, Cursor, and other editors.
- [stitchkit](https://github.com/tygwan/stitchkit) - Claude Code plugin for Stitch and Figma design workflows with skills, agents, and MCP setup.

## Tools and Extensions
